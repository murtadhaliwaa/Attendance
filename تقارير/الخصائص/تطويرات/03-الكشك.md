# 03 — الكشك (Kiosk)

## الوصف
واجهة ميدانية للحضور والانصراف بالوجه، مع تسجيل موظف جديد ورمز طارئ.

## المسارات

| URL | الملف |
|-----|-------|
| `/kiosk` | `app/kiosk/page.tsx` |
| `/kiosk/checkin` | `app/kiosk/checkin/page.tsx` |
| `/kiosk/checkout` | `app/kiosk/checkout/page.tsx` |

## المكوّنات

| الملف | الدور |
|-------|-------|
| `kiosk-scanner.tsx` | الماسح الرئيسي (~1000 سطر) |
| `kiosk-nav.tsx` | `KioskShell` — التخطيط |
| `kiosk-warmup.tsx` | تحميل مسبق للنماذج |
| `app/kiosk/layout.tsx` | Warmup + Shell |

## API المستخدمة

| Endpoint | الغرض |
|----------|--------|
| GET `/api/employees/descriptors` | قائمة البصمات |
| GET `/api/attendance/today` | حالة اليوم |
| POST `/api/attendance/checkin` | حضور |
| POST `/api/attendance/checkout` | انصراف |
| POST `/api/attendance/emergency` | رمز طارئ |
| PUT `/api/employees/descriptors` | موظف جديد |

كلها تتطلب `x-kiosk-key`.

## حالات KioskScanner

`loading` → `scanning` → `processing` → `success` | `error`

## ميزات أمان/دقة
- كاميرا تُطلب بالتوازي مع النماذج
- 3 لقطات متتالية قبل التسجيل
- منع تسجيل وجه مكرر (`findStrictDuplicateMatch`)
- حظر مسبق (حضر مسجّل / انصراف بدون حضور)

## التوسع المستقبلي
- فصل `kiosk-scanner.tsx` إلى: `useKioskCamera` · `useFaceScanLoop` · `KioskEnrollForm`
- انظر `مخالفات-الممارسات` — الملف كبير جداً
