# 06 — التقارير والتصدير

## الوصف
تقارير أسبوعية و per-employee مع تصدير Excel و PDF.

## المسارات

| URL | API |
|-----|-----|
| `/dashboard/reports` | `/api/reports` · `/api/reports/employee` |

## الملفات

| الملف | الدور |
|-------|-------|
| `reports-manager.tsx` | واجهة · فلاتر · تصدير |
| `lib/reports.ts` | استعلامات |
| `lib/report-week.ts` | نطاق التاريخ |
| `lib/report-late-details.ts` | تفاصيل التأخر |
| `lib/report-export.ts` | واجهة تصدير |
| `lib/export-excel.ts` | exceljs |
| `lib/export-pdf.ts` | jsPDF + Amiri |
| `lib/report-export-pdf.ts` | غلاف تقارير PDF |

## الصلاحيات

| إجراء | الصلاحية |
|-------|----------|
| عرض | reports:read |
| تصدير | reports:export (MANAGER فقط) |

## التصدير — عميل فقط

التصدير يعمل في المتصفح (لا API منفصل للملفات) — مقبول للحجم الحالي.

## التوسع

- تقارير كبيرة → API streaming أو خادم يولّد الملف
