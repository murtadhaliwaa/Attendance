# 04 — الموظفون

## الوصف
CRUD للموظفين مع تسجيل الوجه والأكواد والشفتات.

## المسارات

| URL | API |
|-----|-----|
| `/dashboard/employees` | `/api/employees` · `/api/employees/[id]` |
| — | `/api/employees/face-match` |

## الملفات

| الملف | الدور |
|-------|-------|
| `employees/page.tsx` | SSR · `employees:read` |
| `employees-manager.tsx` | قائمة · بحث · إجراءات |
| `employee-form-dialog.tsx` | نموذج إضافة/تعديل |
| `face-enrollment-panel.tsx` | كاميرا + بصمة |

| `api/employees/[id]/route.ts` | GET/PUT/DELETE |

| `lib/employee-validation.ts` | تحقق |
| `lib/employee-codes.ts` | EMP### · emergency |

## حقول Employee المهمة

- `faceDescriptor` — 128 float
- `hasFaceRegistered` — فهرس للاستعلام السريع
- `emergencyCode` — فريد للكشك الطارئ
- `shiftId` · `customEndTime` — توقيت

## صلاحيات UI

| إجراء | الصلاحية |
|-------|----------|
| عرض | employees:read |
| إضافة | employees:create |
| تعديل | employees:update |
| حذف | employees:delete |
| تسجيل وجه | create أو update |
